## Dependencies
* google fonts
* kute.js
* nuxt-gsap-module
* nuxt-sass-resources-loader
* vue-typical
* vuetify
* vue-silentbox
* SnazzyMaps
* SendinBlue
* nuxtjs@/dotenv

## Build Setup
```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
```
